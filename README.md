# VoiceNote+

VoiceNote+ 是一个帮助你快速创建和管理语音笔记的 Visual Studio Code 扩展。

## 功能介绍

- **创建新笔记**：通过命令面板快速创建语音笔记。
- **管理已有笔记**：列出和管理你的所有语音笔记。

## 如何使用

1. 安装扩展：

   - 在 VS Code 扩展商店中搜索 "VoiceNote+" 并安装。

2. 使用扩展：
   - 打开命令面板 (`Ctrl+Shift+P` 或 `Cmd+Shift+P`)。
   - 输入 `VoiceNote: Create Note` 来创建新语音笔记。
